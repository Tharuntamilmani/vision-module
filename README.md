# Jarvis Vision v1

A real-time face recognition system built using OpenCV and InsightFace.

## Features

- Real-time webcam face detection
- Face recognition using deep-learning embeddings
- Multi-image training support
- Unknown face classification
- CPU-based inference support
- Foundation for Jarvis Vision AI

## Tech Stack

- Python
- OpenCV
- InsightFace
- ONNX Runtime
- NumPy
